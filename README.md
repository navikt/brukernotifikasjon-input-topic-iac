# brukernotifikasjon-public-topic-iac

Repo med konfigurasjon for hvem som kan produsere og konsumere fra DittNAV sine åpne topics.

## Hvordan kobler jeg meg på interne-dittnav-topics?

**For skrivetilgang**

```
        - team: myTeam
          application: myApplication
          access: write 
```
**For lesetilgang**

```
        - team: myTeam
          application: myApplication
          access: read 
```
